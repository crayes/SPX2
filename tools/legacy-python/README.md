# Legacy Python (Archived)

Este diretório é reservado para conteúdo legado em Python (ex.: scripts antigos de consumer).

Neste momento, o conteúdo original do repositório estava corrompido (arquivos NUL bytes) e/ou incluía um `venv` versionado.
Por isso, o Python foi removido do caminho de build/deploy para manter o foco total em .NET.

Se futuramente você quiser preservar scripts úteis aqui, mantenha apenas código-fonte e **não** versionar ambientes virtuais (`venv/`).
